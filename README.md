# Data6503-Analysis

This project analyzes the Healthcare Stroke Dataset from Kaggle using exploratory data analysis (EDA) and regression modeling.

The notebook includes:

* Data cleaning and preprocessing
* Visualizations and EDA
* Linear, Polynomial, and Ridge Regression models
* Model evaluation using R², RMSE, and MAE
* Hyperparameter tuning with Grid Search for Ridge Regression

The goal of the project is to explore relationships within the dataset and compare the performance of different regression models for predicting age.
